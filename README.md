#  Minería de Datos 2026

Repositorio académico orientado al aprendizaje y aplicación práctica de técnicas de **Minería de Datos**, como parte de la Tecnicatura en Ciencia de Datos e Inteligencia Artificial.

---

##  Sobre el proyecto

Este repositorio documenta el desarrollo progresivo de habilidades en análisis de datos, desde la exploración inicial hasta procesos de limpieza, transformación y preparación de datos para su posterior análisis.

Cada semana representa un avance en el dominio de herramientas y metodologías clave dentro del flujo de trabajo de un proyecto de datos.

---

##  Contenidos y enfoque

A lo largo del repositorio se trabajan conceptos como:

* Exploración de datos (EDA)
* Limpieza y preprocesamiento
* Transformación de variables
* Manejo de datos faltantes
* Análisis con Python
* Visualización de datos

---

##  Estructura del repositorio

```bash
mineria-de-datos-2026/
```

##  Metodología de trabajo

El desarrollo de las actividades sigue un flujo estructurado:

1. Comprensión del problema
2. Carga y exploración de datos
3. Limpieza y tratamiento
4. Transformación de variables
5. Análisis e interpretación de resultados
