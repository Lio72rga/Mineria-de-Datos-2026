# Semana 10 - Minería de Texto con TensorFlow y Scikit-learn

##  Descripción
En esta semana se trabajó con **técnicas de minería de texto** aplicadas a opiniones en español.  
Se abordaron tanto métodos clásicos (TF‑IDF + Naive Bayes) como modernos (redes neuronales LSTM con TensorFlow), integrando todo el flujo de preprocesamiento, representación y clasificación.

##  Archivos
- `Notebook1_PreprocesamientoTexto.ipynb` → Limpieza, tokenización y stopwords.
- `Notebook2_RepresentacionesTexto.ipynb` → Bag of Words y TF‑IDF.
- `Notebook3_ModelosTensorFlow.ipynb` → Red neuronal LSTM para clasificación.
- `Notebook4_ModelosScikitLearn.ipynb` → Clasificación con Naive Bayes y TF‑IDF.
- `Notebook5_EjercicioIntegrador.ipynb` → Comparación entre Naive Bayes y LSTM.

##  Objetivos
- Aplicar técnicas de **preprocesamiento de texto** para limpiar datos no estructurados.  
- Implementar representaciones numéricas con **BoW y TF‑IDF**.  
- Entrenar un modelo de **Naive Bayes** con Scikit‑learn.  
- Construir una **red neuronal LSTM** con TensorFlow.  
- Comparar el desempeño de modelos clásicos y modernos en clasificación de opiniones.

##  Conclusión
La semana permitió integrar todo el flujo de trabajo en minería de texto:  
- **Naive Bayes**: rápido y eficiente en datasets pequeños.  
- **Red LSTM**: mayor precisión y capacidad de aprendizaje, aunque requiere más datos y tiempo de entrenamiento.  

Este ejercicio integrador muestra cómo elegir la técnica adecuada según el contexto y los recursos disponibles.
